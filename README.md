# MERN Stack Poilerplate using Typescript

## Back End

- Typescript
- Express
- Mongoose
- Cors
- Bcryptjs
- Helmet
- Morgan
- Jsonwebtoken
- Dotenv

### to install the depemdencies run this command

`yarn` or `npm install`

### to run the server

`yarn start` or `npm run start`

### to get the build

`yarn build` or `npm run build`
